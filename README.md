**Aura** is a premium, high-end landing page for a revolutionary AR wearable device. Designed with a futuristic "Neon Glassmorphism" aesthetic, it showcases the perfect blend of hardware and software ecosystem.

---

## ⚡ Key Features

*   **💎 Premium Design**: Sleek dark mode with neon cyan accents and modern glassmorphism.
*   **📱 Responsive Layout**: Fully optimized for Desktop, Tablet, and Mobile devices.
*   **🧩 Component-Based**: Highly modular architecture for easy maintenance and scalability.
*   **✨ Smooth Animations**: Integrated scroll reveals and micro-interactions for an immersive experience.
*   **📡 Modern Tech Stack**: Built with Vite, React, and TypeScript for blazing fast performance.

## 🚀 Getting Started

To get the project up and running locally, follow these simple steps:

### Prerequisites
*   [Node.js](https://nodejs.org/) (v16.0.0 or higher)
*   [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/your-username/tech-product-aura.git
    cd tech-product-aura
    ```

2.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Launch the development server**
    ```bash
    npm run dev
    # or
    yarn dev
    ```
